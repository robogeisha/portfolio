> 🕷 SpideySenses

SpideySenses is a real-time monophonic FM synthesis engine controlled by a simulated elastic web. The engine maps web tension, movement speed, damping, feedback, and rupture-like events to synthesis parameters.

It uses:

  • FM synthesis with tension-controlled modulation depth and pitch deviation.
  • Event detection for short tick, snap, and release gestures.
  • Envelope-based pluck excitation driven by web movement and release speed.
  • Nonlinear feedback shaping for transient pitch and amplitude changes.
  • Parameter smoothing, high-pass filtering, saturation, and output limiting for signal stability.
