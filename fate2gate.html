> 🎲 Fate2Gate

Fate2Gate is a 4-channel stochastic gate sequencer that uses weighted dice to generate drum patterns. Each channel has four dice controlling grid length, start position, gate interval, and timing division.

Dice faces are mapped to rhythmic values. Bias controls how likely each face is to be selected when the die rolls.

> Interface Overview

The interface shows the active dice pair, selected channel, dice values, bias controls, and generated gate pattern. Dice can be rolled manually or updated automatically with Autoroll.

> Interaction

Click a single die to roll it. Click outside the dice to roll the visible dice pair. Enable Autoroll to update the selected channel automatically at cycle boundaries.

> Bias Mechanics

Each die has six faces. Assign each face a rhythmic value and a bias amount. Higher bias makes that face more likely to be selected.

Bias is also used in the dice simulation. Stronger bias makes the die settle more directly toward the weighted face. A more even distribution produces less predictable roll behavior.

> Dice Roles

  • Dice A — Grid Length
    Sets the number of steps in the rhythmic pattern.
  • Dice B — Start Position
    Sets where the pattern begins inside the grid.
  • Dice C — Gate Interval
    Sets the step distance between repeated gates.
  • Dice D — Timing Division
    Sets the timing multiplier or rhythmic subdivision.

Use the Dice Table to edit the six face values for each die.

> Manual / Autoroll

In manual mode, roll dice directly through the interface. In Autoroll mode, reroll the selected channel automatically at cycle boundaries.
