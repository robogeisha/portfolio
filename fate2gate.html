> 🎲 Fate2Gate

Fate2Gate is a 4-channel stochastic gate sequencer that uses weighted dice to generate drum patterns. Each channel has dice controlling grid length, start position, repeat interval, and timing division. Dice faces are mapped to rhythmic values, and user-defined bias controls how likely each value is to occur.

The dice are simulated through a real-time rigid-body physics engine. Strong bias makes the roll feel more locked and stable, while a more even distribution makes the roll more chaotic.

Patterns can be changed manually by rolling dice, or generated continuously with Autoroll, where the system rerolls at cycle boundaries.

> Interaction

Click a single die to roll it manually, or click outside the dice to roll the visible pair. In Autoroll mode, manual rolling is disabled and the selected channel updates automatically as the sequencer runs.

> Bias Mechanics

Each die has six faces, and each face can be made more or less likely with bias controls. Higher bias means that face is more likely to come up when the die rolls.

The bias also affects the physical behavior of the roll. Strong bias makes the die feel more locked and stable, while a more even distribution makes the roll wobblier and more chaotic.

> 🎲 Dice Roles

  • Dice A — Grid Length
    Decides how many steps long the rhythmic pattern is.
  • Dice B — Start Position
    Decides where the pattern starts inside the grid.
  • Dice C — Gate Repetition / Interval
    Sets the step distance between repeated gates.
  • Dice D — Timing Division
    Multiplies or modifies the timing.

Use the Dice Table to edit the six face values for each die.

> Manual / Autoroll

In manual mode, the user performs changes directly by rolling dice. In Autoroll mode, Fate2Gate rerolls automatically at cycle boundaries, turning the pattern into a self-mutating drum gate.
